# omoomo

# [종료] - [미니] 오목 웹 프로젝트

# 🎯 최종 목표

### 기능적 목표

- 소켓 통신을 이용한 웹 오목 게임
- 각자 판단하에 추가 기능(내용) 및 안해본 기술 스택 적용

### 기능외 목표

- 1차 프로젝트 전 조별 사전 팀워크 확인
- 지금까지 배운 내용 복습/응용

### 사용 기술스택

- HTML, CSS, JS, JSP/SERVLET, DB(제한없음)
- 웹소켓
